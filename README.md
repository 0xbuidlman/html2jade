# html2jade #

`html2jade` converts HTML into [Jade](https://github.com/visionmedia/jade) format.

## Install ##

    npm install html2jade

## Usage ##

    html2jade http://twitter.com
    
    html2jade http://twitter.com > twitter.jade
    
    html2jade mywebpage.html > mywebpage.jade
